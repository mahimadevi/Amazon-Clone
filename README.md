# Amazon Clone with MERN Stack

# Project Overview
This project is a complete e-commerce website modeled after Amazon, built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The website includes essential e-commerce functionalities such as user authentication, product management, shopping cart, and order processing.

# Key Features:
1. User Authentication: Secure login and registration using JWT and bcrypt for password hashing.
2. Product Management: Dynamic display of products from the MongoDB database.
3. Shopping Cart: Add-to-cart functionality with real-time updates.
4. Order Management: Ability for users to place orders and view order history.
5. User Profile: Users can view and update their profile information.

# Why This Project is Useful
This project provides a comprehensive example of building a modern web application using the MERN stack. It demonstrates:

1. Full-stack development skills with real-world application.
2. Implementation of secure authentication mechanisms.
3. Handling of complex data relationships in a MongoDB database.
4. Creating a responsive and interactive user interface with React.js.
5. Building RESTful APIs with Node.js and Express.js.
